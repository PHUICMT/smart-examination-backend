# [Linux] 
```bash
python3 -m virtualenv .env #create virtualenv
```
```bash
source .env/bin/activate #activate env
```
```bash
python3 -m pip install -r requirements.txt #install requirements pack
```
```bash
deactivate #deactivate
```
